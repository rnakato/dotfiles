# dotfiles
dotfiles (.emacs, etc.)


```sh
git clone git@github.com:rnakato/dotfiles.git
ln -s ~/git/dotfiles/.conkyrc ~/
ln -s ~/git/dotfiles/.emacs.d ~/
ln -s ~/git/dotfiles/.gitconfig ~/
ln -s ~/git/dotfiles/.jupyter/ ~/
ln -s ~/git/dotfiles/.alias ~/
ln -s ~/git/dotfiles/.pypirc ~/
ln -s ~/git/dotfiles/fish ~/.config/
ln -s ~/git/dotfiles/fisher ~/.config/
echo "source $HOME/git/dotfiles/.bashrc" >> ~/.bashrc
```
